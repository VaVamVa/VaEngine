## Compact Log 2

### [완료] Sky.hlsl V축 반전 수정

**증상**: 하늘 텍스처가 상하 뒤집혀 보임 (위를 보면 지평선색, 아래를 보면 하늘색)

**원인**: `v = phi / PI + 0.5f` → 위쪽 시선(phi=+π/2)일 때 v=1.0 (텍스처 하단) 샘플링

**수정** (`Engine/_Shaders/DirectX/Sky.hlsl`):
```hlsl
// before
float v = phi / PI + 0.5f;
// after
float v = 0.5f - phi / PI;
```
